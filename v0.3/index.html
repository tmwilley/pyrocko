Pyrocko Manual¶

Pyrocko is open source toolbox and library for seismology.

Features¶

seismogram browser and picker waveform archive management waveform processing Green's function calculation and management seismic ray theory synthetic seismograms online catalog access FDSN web service access beachball plotting online topo data access enhanced GMT mapping geodesic functions
